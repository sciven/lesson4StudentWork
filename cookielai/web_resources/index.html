What does SPARC stand for?

SPARC is a acronym for: Solving Problems with Algorithms

Why?

I made my SPARC for tracking:

  • eating fruits and veggies
  • exercising
  • reducing waste

How is it Used?

The SPARC device is used for trcking things. So like General Health and stuff like that. You would press a button, so like button #1 = eating fruits and veggies or something like that, each button equals something different here is an example:

  • Button #1 = eating fruits and veggies
  • Button #2 = exercising
  • Button #3 = reducing waste

So each time you press a button on your SPARC device, when you load it on to a computer it will show as 1.
